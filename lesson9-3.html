【核心素养与学习目标】

  • 对比与推理：对比 Na₂O 与 Na₂O₂ 的性质差异[cite: 1]；掌握 Na₂CO₃ 与 NaHCO₃ 的水溶性、热稳定性及与酸反应剧烈度的异同[cite: 1]。
  • 生活与工业：掌握氢氧化钠（NaOH）易潮解与吸收 CO₂ 变质的检验方法[cite: 1]；了解氯碱工业及硝酸钾（KNO₃）作为化肥/火药原料的用途[cite: 1]。
任务一：钠的氧化物对比 —— Na₂O 与 Na₂O₂
对比项目 氧化钠 (Na₂O) 过氧化钠 (Na₂O₂)
1. 颜色与状态 白色固体[cite: 1] 淡黄色固体 [cite: 1]
2. 氧元素化合价 -2 价 -1 价
3. 与 H₂O 反应方程式 Na₂O + H₂O → 2NaOH 2Na₂O₂ + 2H₂O → 4NaOH + O₂↑
4. 与 CO₂ 反应方程式 Na₂O + CO₂ → Na₂CO₃ 2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂
5. 特殊用途 化工原料 呼吸面具与潜水艇中的 供氧剂 及漂白剂
任务二：氢氧化钠（NaOH）的潮解与变质检验

1. 潮解与变质：氢氧化钠俗称烧碱、火碱、苛性钠[cite: 1]，极易吸水而潮解，在空气中吸收 CO₂ 变质的化学方程式为： 2NaOH + CO₂ → Na₂CO₃ + H₂O [cite: 1]。

2. 变质检验方案设计：

• 方案一（加酸法）：
取少量样品配成溶液，滴加足量稀盐酸[cite: 1]。若观察到有气泡（CO₂）产生，说明 NaOH 已经变质[cite: 1]。
• 方案二（沉淀法）：
取少量样品配成溶液，滴加 CaCl₂ 或 BaCl₂ 溶液[cite: 1]。若产生白色沉淀（CaCO₃ 或 BaCO₃），说明已经变质[cite: 1]。

性质 碳酸钠 (Na₂CO₃ / 纯碱 / 苏打) 碳酸氢钠 (NaHCO₃ / 小苏打)
1. 水溶性与 pH 易溶于水，水溶液碱性较 强 (pH 较大)[cite: 1] 微溶/溶解度较 小 ，水溶液呈弱碱性[cite: 1]
2. 热稳定性 受热 不分解 [cite: 1] 受热易分解： 2NaHCO₃ --Δ--> Na₂CO₃ + H₂O + CO₂↑ [cite: 1]
3. 与 HCl 反应剧烈度 反应相对平缓（分步进行：CO₃²⁻ → HCO₃⁻ → CO₂）[cite: 1] 反应极 剧烈 （瞬间产生大量 CO₂）[cite: 1]
任务四：转化与应用

1. 除杂方案：若除去 Na₂CO₃ 固体中混有的 NaHCO₃ 杂质，最简便的方法是： 加热灼烧固体至不再产生气体 [cite: 1]。

2. 硝酸钾（KNO₃）：硝酸钾易溶于水，溶于水时 吸热 （吸/放热）[cite: 1]。受热分解放出氧气：2KNO₃ --Δ--> 2KNO₂ + O₂↑[cite: 1]，常用于制造黑火药和农用复合化肥[cite: 1]。

🧠 9.3 钠和钾 Important Compounds 知识结构图
9.3 钠和钾 Important Compounds +
  • 氧化物与氢氧化物 +
    • Na₂O：白色固体；Na₂O₂：淡黄色强氧化剂

    • NaCl/KCl：调味、化工原料、食盐晒制
    • KNO₃：强氧化性、火药、化肥
